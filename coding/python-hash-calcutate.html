使用python计算文件的hash值并验证

发表于 2017-03-05 | 分类于 编程 |

最近在学习python编程语言，正好需要计算文件hash值的工具，虽然网上有这样的软件可以完成这个工作，但本着自己动手，丰衣足食的态度，应用python现编一个程序，纯当作编程练手。

使用方法

在需要计算hash值文件的目录下打开命令提示符，键入以下命令：
python hash_caculator.py 文件名
本文使用环境为python3 on windows，请确保系统中系统中已经安装有python3。
计算完成后，命令提示符下会提示你选择hash算法，本例中提供“md5”，“sha1”，及“sha256”等算法。
输入选择的算法后，会提示输入用于验证的哈希值，输入并回车后，屏幕上就会显示出验证结果，并且把结果写入同一目录下名为“hashoutput.txt”的文件。
